LampShop — микросервисы

Учебный пример на порту 8000 с путём /items — это другой проект. Здесь каталог и заказы разнесены по ТЗ:

Product Service (порт 3001)
  • Swagger UI — /docs
  • ReDoc
Order Service (порт 3002)
  • Swagger UI — /docs (сначала Authorize → X-Session-Id)
  • ReDoc
Быстрый старт
  1. В каталоге product-service: python -m app.seed (один раз, чтобы были 20 товаров).
  2. Запустите оба сервиса (см. run-dev.ps1 в корне репозитория) или вручную: 
    cd product-service
python -m uvicorn app.main:app --reload --host 127.0.0.1 --port 3001
    cd order-service
python -m uvicorn app.main:app --reload --host 127.0.0.1 --port 3002
  3. В Order Service Swagger: Authorize → введите UUID сессии, скопируйте id товара из Product Service → POST /api/v1/cart/items → POST /api/v1/orders.

Коллекция Postman: LampShop_API_Postman.json в корне проекта.
